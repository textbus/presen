支持多人协作的 PPT 编辑器
============================================

大家好，我是画笔，Textbus 富文本编辑器的作者，一直以来我都专注于协作文档、表格及 PPT 办公三件套的研发。 目前 Textbus 富文本编辑器的框架基本已经成型，并且我也开发了 XNote 富文本编辑器给大部分用户开箱即用。

鉴于开源项目中我暂时没有找到**完善且能够自由使用**的多维表格和 PPT 编辑器项目，本着对技术的热爱以及自身的积累，想借着我最后的热情把这两个项目逐步完善，但由于精力有限，我也不确定哪一个的更被大家需要，欢迎大家点击 star，我将通过项目的关注人数调整开发的优先级。

由于近期我也是待业状态，当前企业的需求对于协作文档的开发者并不算友好，我近期也暂时没有找到办公三件套相关的工作，所以我可以有较多的业余时间来完成项目的迭代。如果有老板愿意赞助项目的开发，帮我减轻经济压力，我感激不尽，为不影响阅读，赞助的二维码我放在本页的最后。

## PPT 项目规划及目标

PPT 我目前所掌握的其 160 多种基础图形绘制，包括调整点，变换规则，编组等，同时也基本理清并实现了 Master/Relationship/layout/media 以及 Slide 中节点身的的 Props 等的绘制。

由于 PPT 细节很多，还包括动画，转场等效果。我实现了一部分，但还有很多并未研究清楚，需要花费大量的时间去查找相关文件以及实地试验，所以我不能保证真的能复刻 office 中 ppt 的所有功能。

在我当前的实现中，我是用 svg 去实现的 ppt 图形绘制，但 svg 有较多的局限，在此开源项目中，我应该会采用 canvas 来实现，以规避 svg 中一部分功能不能实现的问题。

### 功能清单

- [x] 160 多种图形的绘制和调整编辑（已实现）
- [x] 图形边框，含多线及不同类型的虚线（已实现部分）
- [x] 图形阴影（已实现）
- [x] 文字横排竖排及各种对齐
- [x] 图形编组及编组后调整
- [ ] 图形 3D 效果
- [ ] 连线及路径规避及吸附
- [ ] 参考点计算
- [ ] 表格（已实现部分）
- [ ] 其它多媒体功能（未研究）
- [ ] 转场动画（实现了一小部分）
- [ ] 播放动画（实现了一小部分）
- [ ] smartArt
- [ ] 图表

### 其它相关核心需求

- [x] PPTX 的解析（完成了大部分）
- [x] 多人协作（因建模采用的 Textbus 内核，默认支持协作）
